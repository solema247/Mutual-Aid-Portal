# Quick Video Recording Guide - 5 Minutes

**Goal:** Record a simple 5-minute demo showing both user perspectives

---

## ⚡ Super Quick Setup (2 minutes)

### 1. Install Loom (Easiest Option)
```
1. Go to: https://www.loom.com/download
2. Install desktop app or Chrome extension
3. Sign up (free account)
4. You're ready!
```

### 2. Prepare Your Browser
```
1. Open: http://localhost:3001
2. Login with your credentials
3. Close all other tabs
4. Set browser zoom to 100%
5. Have script open on phone or print it
```

### 3. Start Recording
```
1. Open Loom
2. Click "Record"
3. Select "Screen + Camera" or "Screen Only"
4. Select your browser window
5. Click "Start Recording"
6. Follow script below
```

---

## 🎬 5-Minute Script (Word-for-Word)

### SCENE 1: Intro (15 seconds)
**[Your face on camera, or just start screen recording]**

Say:
> "Hi! I'm going to show you how our compliance screening works. First, from an ERR Program Manager's view uploading an F1, then from a Finance Officer's view reviewing and approving. Let's start."

**[Click to screen share / start screen recording]**

---

### SCENE 2: Program Manager Upload (90 seconds)

**[Navigate to: http://localhost:3001/err-portal/f1-work-plans]**

Say:
> "I'm logging in as a Program Manager. I need to submit an F1 work plan."

**[Click "Direct Upload" tab]**

Say:
> "I select my state and emergency room, then upload my F1 PDF with ID documents attached."

**[Select: State = "Khartoum", Room = "ERR-KH-001", Upload any PDF file]**
**[Click "Upload & Process"]**
**[Wait for processing]**

Say:
> "The system extracts the data with OCR. Behind the scenes, it's also finding the ID documents in my PDF and extracting beneficiary information. I review, confirm, done."

**[Click "Confirm Upload" when ready]**
**[Success message appears]**

Say:
> "That's it from my side. Nothing extra for compliance - it's all automatic."

---

### SCENE 3: Compliance Dashboard (2 minutes)

**[Navigate to: http://localhost:3001/err-portal/compliance]**

Say:
> "Now I'm a Compliance Officer. This is my dashboard - I can see all pending screenings."

**[Point to statistics cards]**

Say:
> "Three pending, five cleared, two flagged, one rejected. Let me check a pending case."

**[Click "View" on Fatima Hassan Ibrahim (pending)]**

Say:
> "Here's Fatima, a Finance Officer. I can see her ID info extracted from the document. I'll run the screening."

**[Click "Run OFAC Screening Check"]**
**[Wait 2 seconds]**

Say:
> "No match found. I'll mark her as cleared."

**[Type in notes: "No OFAC match. Cleared."]**
**[Click green "Clear" button]**
**[Close modal]**

Say:
> "Done. Now let me show you a flagged case."

**[Filter by "Flagged" status]**
**[Click "View" on Omar Abdullah Khalil]**

Say:
> "This person was flagged - 87% name match with OFAC list. But look: different birth date, different ID number. False positive. I can clear it with justification."

**[Close modal - don't need to complete]**

Say:
> "And here's a rejected case - exact OFAC match. Payment is blocked."

**[Filter by "Rejected"]**
**[Click "View" on Hassan Mohamed Al-Bashir]**
**[Show the rejection details]**

Say:
> "This person is on the sanctions list. The system has blocked this F1. ERR will need to resubmit with a different person."

**[Close modal]**

---

### SCENE 4: F1 Review Integration (90 seconds)

**[Navigate to: http://localhost:3001/err-portal/f1-work-plans]**
**[Click "ERR App Submissions" tab]**

Say:
> "Now I'm a Finance Officer reviewing F1 submissions. Notice the new screening column - green, yellow, orange, red badges."

**[Point to screening column]**

Say:
> "Let me open a project."

**[Click "View" on any project with screening data]**

Say:
> "Here's the standard F1 detail view. But now there's a screening tab."

**[Click "Screening" tab]**

Say:
> "I can see all beneficiaries and their status. Two cleared, one pending. I get a warning if there are issues."

**[Show the beneficiary list]**

Say:
> "If everything is cleared, I approve as normal. If there's a flag, I check with compliance first. If rejected, I can't approve at all."

---

### SCENE 5: Wrap Up (30 seconds)

**[Show compliance dashboard or F1 table]**

Say:
> "So that's it. Program Managers upload F1s as usual - zero changes. Compliance Officers get a dedicated dashboard for screening. Finance Officers see status in their review workflow. We get 100% OFAC coverage with zero disruption to our 1-3 day timeline. Thanks for watching!"

**[Stop recording]**

## 🆘 Quick Help

**Recording not working?**
- Try QuickTime on Mac (Cmd+Shift+5)
- Try Windows Game Bar (Win+G)

**Video too long?**
- Focus on Scene 3 & 4 only (compliance dashboard + F1 integration)
- Skip Scene 2 (F1 upload) if time is short

**Audio poor?**
- Use your phone earbuds as microphone
- Record in quiet room

**Need help?**
- See full guide: `USER_JOURNEY_DEMO.md`
- Or just take screenshots and present live
